| 译者注：ECMAScript提案分三个阶段：草图(Sketch) -> 提案(Proposal) -> 规范(Standard)，所以Async很大的希望能成为规范.

Async 函数其他的一些用法

现在存在如下的async函数的变体：

  • Async 函数声明： async function foo() {}
  • Async 函数表达式： const foo = async function() {}
  • Async 方法定义： let obj = { async foo() {} }
  • Async 箭头函数：const foo = async () => {};
