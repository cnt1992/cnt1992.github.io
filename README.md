## Install

1.Clone

```
git clone git@github.com:cnt1992/cnt1992.github.io.git cnt1992
```

2.Initial Submodule

```
cd cnt1992
git submodule init
git submodule update
```

3.Install Dependencies

```
npm i -d
```

4.Serve

```
hexo s --watch
```

> Preview Href: [http://localhost:4000](http://localhost:4000)
